BookMaker
=========

ACM-MISIS Team Reference Document
---------------------------------

BookMaker searches directory for *.h and *.cpp files. It combines *.h files into HEADER and compiles every *.cpp file with this HEADER. 

Usage
-----

javac BookMaker.java
java BookMaker <dir>
